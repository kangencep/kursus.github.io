MATERI KURSUS KODING

HTML

Bahasa yang digunakan untuk membuat kerangka situs web Anda.

CSS

Bahasa yang digunakan untuk membuat dan mendesain tampilan situs web Anda.

JAVA SCRIPT

Bahasa pemrograman tingkat tinggi, dinamis, populer di internet.

BOOTSTRAP

Kerangka kerja CSS yang sumber terbuka dan bebas untuk merancang situs web dan aplikasi web.

PHP

PHP adalah bahasa skrip yang dapat ditanamkan atau disisipkan ke dalam HTML

MYSQL

MySQL adalah sistem manajemen database relasional open source (RDBMS) dengan client-server model.

CODEIGNITER

CodeIgniter merupakan aplikasi sumber terbuka yang berupa kerangka kerja PHP dengan model MVC.

LARAVEL

Laravel adalah kerangka kerja aplikasi web berbasis PHP yang sumber terbuka, Model-View-Controller.
